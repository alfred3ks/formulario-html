Input de tipo radio button:

Como los tres input llevan el mismo name estan relacionados.

El atributo value es el valor que se usara para recogerlo por JS, recogera genero: masculino, genero:femenino,genero:desconocido

Input de tipo radio checkbox:

Son muy parecido a los de tipo radio con la diferencia que los checkbosx nos permite seleccionar de una lista relacionada

Aqui igual vemos el name con sus repectivos value, ambos atributos son importantes para poder manejar con JS

Input de tipo select:



El value de la etiqueta option es el valor de su contenido.

Podemos poner una option con el atributo select por defecto

Si colocamos el atributo multiple la seleccion de los valores puede ser multiple

Vemos otro tipo de select: algo diferente.

Usaremos la etiqueta optgroup para agruparlos.

Aqui el interesante es el atributo label para crear los subgrupos

Los datalist

Otro elemento de lista que llego con HTML5

incluyen un filtro para buscar informacion mas sencilla

Para este tipo de lista debemos suprimir o bien el value o el texto que tiene la option dependera lo que ussemos.

Al ser un input podemos usarlo como filtro al poner una letra nos mostrara los elementos que incluyan esa letra.
